# Knowledge Ingest Drop Zone

Put your Class 11 Physics NCERT source files here.

Accepted file formats:
- .txt
- .md
- .json
- .pdf (supported if pdf-parse is installed)

Recommended naming format:
- 11_Physics_NCERT_Chapter01_UnitsAndMeasurements.md
- 11_Physics_NCERT_Chapter02_MotionInAStraightLine.md

Then run from backend folder:

npm.cmd run ingest:knowledge

After ingest completes, restart backend and ask tutor questions.
